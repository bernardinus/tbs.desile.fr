Characters List

  • You can check the "Have ?" column to indicate you have that character
  • You can tell what job level he has
  • The first line tells you how many characters you have
  • The "P.o.T" and "P.o.F" tells you if a character can be obtained by Pact of Fellowship or the Pact of Truth

Item List

  • You can tell you how many items you have
  • The column "Currently you need" takes into account the character you have and which job they have to tell you how many more item you need to get to Job Level 3
  • The column "For all characters, you would need :" tells you how many items are needed to get every character to Job Level 3

Stages List

  • The last column (with the pencil) opens a dialog to enter stats about a stage (experience and coins)

You can add either an experience gained or coins acquired in a stage; theses stats will then be averaged on the stage list page so that you can tell which stage gives the more experience, which stage is the most efficient given stamina, etc.
